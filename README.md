### CSE360 Computer Interface

Computer Interface is a course at my university. This course teaches us about the interface we use in our day-to-day life with computers. In this course, we have to complete a group project in which we implement the interfaces we learned in theory and labs. For this reason, 3 of my groupmates and I worked on a project to build a **Driver Drowsiness Detection System**. I have added the report of this project, which helps to understand how the system will work, and also there are some diagram which helps to understand the work of the system more easily.
